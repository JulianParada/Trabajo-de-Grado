¿Qué encuentras en GAT Gate?

Mapas de Gobernabilidad Kumu

Kumu es una plataforma web cuya especialidad es el mapeo de las relaciones, aqui podrás encontrar mapas interactivos y dinámicos para mostrar de forma fácil y clara las relaciones entre interesados, sistemas, comunidades, proyectos, plantas, entre otros. Actualmente es utilizada por el proyecto de fortalecimiento institucional de GAT para la visualización del gobierno corporativo y la estructura interna del ecosistema tanto para personal, como para proyectos e incluso para el mapeo de objetivos y tareas

Monitoreo Global del Proyecto

El Dashboards principal del GAT Gate es una herramienta para el monitoreo del estado actual del proyecto GAT, esto es un mecanismo visual para la representación de información de una manera amigable con el usuario. La información que se presenta es variada y está relacionada a datos sobre actividades diarias, porcentajes de avances de objetivos, estadísticas sobre procesos o resultados, entre otras. Estos datos se obtienen a través de diferentes repositorios que guardan esta información, por ejemplo, OSF.

¿Eres miembro de GAT Gate?

Si eres miembro activo de GAT o estas relacionado con este proyecto tienes la posibilidad de conocer de forma detallada la información de los indicadores presentados en el tablero de monitoreo global del proyecto. Inicia sesión dando click en el botón que aparece abajo o a través de la barra de navegación usando las credenciales asignadas, si aún no las tienes, solicítalas con el director del proyecto al que perteneces.
